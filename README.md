基于Create-React-App创建的单页面富文本编辑器（tinymce）

## 可用命令

在项目文件中支持Create-App-React的脚本：
```
yarn start
yarn test
yarn build
yarn eject
yarn build
```

## tinymce

所需依赖以及模块化导入tinymce

```json
{
"dependencies": {
    "@tinymce/tinymce-react": "^3.5.0",
    "tinymce": "^5.2.0"
  }
}
```
